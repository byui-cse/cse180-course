CSE 180: Week 12

Ponder and Prove

Your Tasks

Goal this week: Explore how to use calculation to assess and classify compression algorithms by their efficiency.

You are given these descriptions and a one- or two-word jargon label for different but related types of algorithms:

  1. One constant operation reduces the problem size by one — Linear.
  2. One pass through the input reduces the problem size by one — Quadratic.
  3. One constant operation reduces the problem size by half — Logarithmic.
  4. One pass through the input reduces the problem size by half — Linear Logarithmic.

You have two tasks this week. The first is to analyze the text compression algorithm you learned in class and determine which of these four types best describes it. The second is to examine the algorithms you have collected in your interviews with your loved ones and aquaintances. Is there any algorithm they were using that could be improved by applying the algorithm you saw in class to do text compression? The algorithm can be used to do things other than text compression.😀

How to Fulfill Return-and-Report Requirements

  • Make sure you take notes on the analysis you do outside of class.
  • Prepare to share with your classroom team your notes and discoveries about the algorithm you assessed.
